# Outros Comandos

### Ping

Para ver o ping da Kokia, use:

```text
k!ping
```

### Prefixo

Para ver o prefixo da Kokia, use:

```text
k!prefix
```

Para mudar o prefixo da Kokia, use:

```text
k!prefix <prefix>
```
