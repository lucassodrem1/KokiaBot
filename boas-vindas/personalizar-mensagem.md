# Personalizar mensagem

### Personalizar mensagem

Para alterar o título, use:

```text
k!welcome title <mensagem>
```

Para alterar a descrição, use:

```text
k!welcome desc <mensagem>
```

Para alterar o footer, use:

```text
k!welcome footer <mensagem>
```
